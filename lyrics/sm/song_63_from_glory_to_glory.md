Pese 63: From Glory to Glory

[Intro]

[Verse 1]
le pouliuli hele mai, le malamalama holo,
le Atua nui, le faʻamoemoe polo,
le faʻatuatua paʻa, le alofa mau,
le lagi ao, le lalolagi hau.

[Verse 2]
le gata hele mai, le agasala lawe,
le oti kokoke, le Atua hawe,
le faaolataga hele mai, le faʻamoemoe ola,
le faʻatuatua paʻa, le Faaola hola.

[Chorus]
le malamalama e hele mai, le pouliuli pau,
le Atua mālamalama, le faʻamoemoe hau,
le faaolataga kokoke, le faʻatuatua ola,
le Faaola nui, le ola hola!

[Verse 3]
le ana pōhaku, le afi mehana,
le vai ola, le ola banana,
le agelu kiaʻi, le Atua mau,
le folafolaga paʻa, le alofa hau.

[Verse 4]
Atamu uē, Eva kaumaha,
le Atua lohe, le folafolaga aha,
le faaolataga kokoke, le Faaola mai,
Parataiso hoʻi, le faʻamoemoe kai.

[Chorus]
le malamalama e hele mai, le pouliuli pau,
le Atua mālamalama, le faʻamoemoe hau,
le faaolataga kokoke, le faʻatuatua ola,
le Faaola nui, le ola hola!

[Bridge]
le oti pau, le ola ola,
le pouliuli holo, le malamalama hola,
le Atua mau, le alofa aloha,
le faaolataga kokoke, le lagi ola.

[Outro]
le malamalama mau... le pouliuli pau...

Weedduu 52: Eden's Memory

[Intro]

[Verse 1]
Jalqaba irratti Waaqayyo uume,
Waan hunda haaraa, waan turee hin jiru,
Addaamii fi Hewaan Jannataa keessa,
Kufaatii dura, gatii dura.

[Verse 2]
Sana booda cubbuun fi du'aan fi dhukkubbiin dhufe,
Garuu Waaqayyo bu'aa fayyina karoorse,
Dhaloota hunda keessa, waggaa hunda keessa,
Dhiiga keessa fi imimmaan keessa.

[Chorus]
Jalqaba hanga dhumaatti, karoora Waaqayyoo mul'ata,
Waan himamamu caalaa bareeda,
Alfaa fi Omegaa, Jalqabaa fi Dhumaa,
Fuulduraa fi darbe walitti hidha!

[Verse 3]
Raajoonni waa'ee Isa dhufuu dubbatan,
Ilma tokkicha Abbaa,
Yeroo guutuu keessatti,
Waaqayyoo irraa bu'e.

[Verse 4]
Fannoo fi awwaalaa fi du'aa ka'uu,
Mo'icha cubbuu, guutummaatti,
Ammaan akka dhufu eegna,
Ijoollee Isaa nagaadhaan gara manaatti fiduuf.

[Chorus]
Jalqaba hanga dhumaatti, karoora Waaqayyoo mul'ata,
Waan himamamu caalaa bareeda,
Alfaa fi Omegaa, Jalqabaa fi Dhumaa,
Fuulduraa fi darbe walitti hidha!

[Bridge]
Waan biqiltuu bareeda keessatti jalqabe,
Waaqayyo bakka hundatti yeroo jiru xumurama,
Samiin haaraan fi lafti haaraan irra deebi'anii dhalatu,
Ganama bara baraa sana.

[Outro]
Jalqaba hanga dhumaatti... Inni amanamaa dha, Waaqa keenyaa fi michuu keenya...
